# API Testing – Portfolio 🌐

## 📌 Overview

This project contains API tests for Fake Store API.

## 🔧 Tools

* Playwright (API testing)
* TypeScript

* GET /products
* POST /products
* DELETE /products
* Validation & edge cases

```bash
npx playwright test
```
