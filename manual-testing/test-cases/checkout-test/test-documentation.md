# 💳 Checkout – Test Scenarios (SauceDemo)

## 📌 Overview

The goal is to verify the correctness and reliability of the checkout process, including form validation, order summary, and successful order completion.

---

## 🧪 Scope

* Checkout form validation (First Name, Last Name, Postal Code)
* Navigation between checkout steps
* Order summary (items, prices, totals)
* Completing the purchase flow
* Canceling checkout

---

## 🧾 Notes

* Each test case should include screenshots where applicable
* Focus on validation messages and edge cases
* Verify consistency between cart and checkout overview
* Pay attention to total price calculation (items + tax)
* Ensure proper navigation between steps (Back / Cancel / Continue)
